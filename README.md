# UrbanGo — App de Transporte Urbano

Aplicação desenvolvida como atividade da disciplina de Desenvolvimento Mobile, alinhada ao **ODS 11 — Cidades e Comunidades Sustentáveis**, com foco na meta **11.2**: “Proporcionar acesso a sistemas de transporte seguros, acessíveis e sustentáveis”.


 1. Descrição do Problema

A mobilidade urbana é um desafio recorrente nas cidades brasileiras. Informações sobre horários e rotas de ônibus costumam estar dispersas, desatualizadas ou acessíveis apenas em pontos físicos. Isso causa atrasos, dificulta o planejamento do deslocamento e reduz a qualidade de vida dos usuários do transporte público.

O **UrbanGo** surge para centralizar informações básicas de transporte urbano, oferecendo ao usuário uma forma simples, rápida e acessível de consultar linhas e horários.

 2. Justificativa

A falta de informações organizadas sobre transporte público afeta especialmente estudantes, trabalhadores e moradores de regiões periféricas.
O desenvolvimento deste aplicativo contribui para o **ODS 11**, ao apoiar o acesso a sistemas de transporte mais eficientes e inclusivos.

Além disso, o projeto permite aplicar práticas de engenharia de software, UX e acessibilidade (WCAG 2.1), simulando um cenário real de desenvolvimento.

 3. Público-Alvo

* Usuários que dependem de transporte público para estudar ou trabalhar.
* Moradores de cidades pequenas que não possuem app oficial.
* Pessoas que desejam consultar rapidamente linhas e horários.

 4. Objetivos do Aplicativo
 Objetivo Geral

Desenvolver um aplicativo móvel simples (PWA) que facilite o acesso às informações básicas de transporte urbano.

 Objetivos Específicos

* Exibir lista de linhas de ônibus.
* Mostrar horários e principais paradas de cada linha.
* Permitir ao usuário favoritar linhas.
* Aplicar boas práticas de usabilidade e acessibilidade.
* Entregar um MVP funcional com documentação clara.

5. Tipo de Aplicação

**PWA — Progressive Web App**

* Instalável pelo navegador.
* Compatível com desktop e mobile.
* Sem necessidade de loja de aplicativos.
  Tecnologias utilizadas: **HTML, CSS, JavaScript**.

6. Instalação e Uso

1. Acesse o arquivo `index.html` no navegador.
2. Para instalar no celular:

   * Android (Chrome): Menu → “Adicionar à tela inicial”.
   * iOS (Safari): Compartilhar → “Adicionar à tela de início”.
3. Use os menus para navegar entre:

   * Linhas
   * Detalhes da Linha
   * Favoritos

7. Requisitos do Sistema

* Navegador moderno (Chrome, Edge, Firefox, Safari).
* Suporte a Service Workers (para instalação offline).
* Android ou iOS atualizado.
